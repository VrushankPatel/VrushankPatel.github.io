Complexica

Colorize your old black and white photos.

  • URL : complexica.app
  • Category : Web Application (Deep Learning and image processing at server side).
  • Technologies:Python (Flask, OpenCV, Numpy, Caffe DNN), Firebase, Nginx (API Gateway), ReactJS (Web-client), Docker & Heroku (Deployment).

Complexica works like a magic. It just convert your black and white photo into coloured one in just a few clicks. The images look like they were taken using a digital camera. Complexica uses deep learning model to transform the old B/W image into beautiful colored one, this model is trained to add color in the black and white photo based on photo's black and white color intensity, ratio and exposure.

Eduwiz

School management System.

  • URL : Eduwiz.app
  • Category : Web Application.
  • Technologies:Django-Python, Javascript (JQuery, ChartJS), ARC4(Encryption).

Eduwiz is a collection features, specially designed to manage the day-to-day administrative tasks of schools. it allow schools to digitally monitor the daily activities along with managing all the resources and information on a single platform. It provides an efficient software solution, developed to serve the academic institutes, also able to handle the fee management, student/staff record management, attendance management, timetable scheduling, homework management, exam data management easily.
